# 🪖 Smart Helmet with Accident Detection & GPS Alert

## 📖 Overview

Every second counts after a road accident.

The **Smart Helmet with Accident Detection & GPS Alert** is an **IoT-based intelligent safety system** that automatically detects accidents and immediately sends the rider's live GPS location to emergency contacts. The project minimizes rescue time, improves emergency response, and enhances road safety.

---

## ❗ Problem Statement

🚨 Thousands of accident victims lose valuable time because:

- Emergency services are not informed immediately.
- Victims may be unconscious.
- Exact accident location is unknown.
- Delayed medical assistance increases fatalities.

---

## 💡 Our Solution

The smart helmet continuously monitors the rider using sensors.

If an accident is detected:

✅ Detects the impact automatically

✅ Retrieves the current GPS location

✅ Sends an emergency SMS

✅ Alerts family or emergency contacts

✅ Enables faster rescue operations

## 🛠️ Hardware Components

| Component | Purpose |
|-----------|----------|
| Arduino UNO / ESP32 | Main Controller |
| MPU6050 | Detects Accidents |
| GPS Module (NEO-6M) | Live Location |
| GSM Module (SIM800L) | Sends SMS Alerts |
| Power Supply | System Power |
